## Install
Need to install [Python](https://www.python.org/downloads/) 3.7.2+ first

Make example directory
```shell
> mkdir web3py-ext-examples & cd _$
```
Set python virtual environment (preferable but **optional**)
```shell
> python -m venv .venv
> . ./.venv/bin/activate
```
Install
```shell
> pip install web3py-ext
```
